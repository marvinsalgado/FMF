Evaluación Desarrollador Jr.
Bienvenido a tu prueba de Desarrollador Jr. de la Federación Mexicana de Fútbol.
Estas son las actividades que debes realizar:
  1. Utilizar Angular 7 o superior.
  2. Instalar la librería ng-wizard https://www.npmjs.com/package/ng-wizard para crear un componente por pasos.
  3. En el primer paso debes mostrar un formulario de datos generales con la siguiente información:
    • Nombre
    • Apellido Paterno
    • Apellido Materno
    • Fecha de Nacimiento (Con un campo de selección de fecha)
    • Género *
    • Nacionalidad *
    • Club de Fútbol *
    • R.F.C. **
    • Ocupación
    • Botón para enviar información
    * Estas opciones deben alimentarse de manera dinámica utilizando los JSON que se encuentran en la carpeta "data", también se requiere crear los modelos de datos y guardarlos en la carpeta "models"
    ** Este campo solo debe aparecer si el registro que se captura es de alguien mayor a 18 años, en caso que lo sea, validar la estructura del R.F.C. antes de enviar la información
  4. Utilizando la información capturada en el formulario, dentro del paso dos mostrar una ficha similar a la siguiente (Se tomará en cuenta el diseño implementado):
  5. Instalar la librería https://www.npmjs.com/package/jspdf para crear PDF.
  6. En el tercer paso del componente, desplegar la misma información del formulario del primer paso en un PDF.
  7. Una vez hayas concluido tu evaluación, debes subirlo a tu propio repositorio de GitHub o GitLab y enviar el link para su revisión.
